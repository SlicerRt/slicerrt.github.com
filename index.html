Welcome to SlicerIGT

Collection of Slicer modules to supplement image guided therapy procedures. In order to begin, checkout the root level repository (https://github.com/SlicerIGT/SlicerIGT) and run CMake.

$ cd SlicerIGT/SlicerIGT
$ git fetch origin
$ git checkout master
