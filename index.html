Welcome to SlicerIGT

Collection of Slicer modules to supplement image guided therapy procedures. In order to begin, checkout the root level repository (https://github.com/SlicerIGT/SlicerIGT) and run CMake. SlicerIGT is only supported under Slicer4.

Installing SlicerIGT

The SlicerIGT extension is present in the extension manager. To use SlicerIGT modules, simply install from the extension manager.

Developing SlicerIGT

Fork the module you wish to develop. Perform any modifications you wish and simply make a pull request and we will review your changes!

To set up the development environment, you will need a set of tools for development. The following commands assume a properly installed command line git.

$ mkdir <root>/SlicerIGT
$ mkdir <root>/SlicerIGT/SlicerIGT
$ cd <root>/SlicerIGT/SlicerIGT
$ git clone https://github.com/SlicerIGT/SlicerIGT.git

Once cloned, use CMake to produce the build folder. Depending on development environment, run make/VisualStudio build/etc...
