3D Slicer extension for image guided therapy

Welcome to SlicerIGT

Collection of Slicer modules to supplement image guided therapy procedures.

Installing SlicerIGT

The SlicerIGT extension is present in the extension manager. To use SlicerIGT modules, simply install from the extension manager.
